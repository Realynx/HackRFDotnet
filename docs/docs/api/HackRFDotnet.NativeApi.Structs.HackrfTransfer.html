USB transfer information passed to RX or TX callback. A callback should treat all these fields as read-only except that a TX callback should write to the data buffer and may write to HackRFDotnet.NativeApi.Structs.HackrfTransfer.valid_length to indicate that a smaller number of bytes is to be transmitted.
